二、服务器端#

1.使用工具#

2.部署代码#

下载得到frp_0.44.0_linux_amd64.tar.gz 在服务端中解压获得文件中包括 frps frps.ini

frps是监听程序 frps.ini 是服务端参数

[common]
bind_addr = 0.0.0.0
bind_port = 10000
bind_udp_port = 10001
kcp_bind_port = 10002
vhost_http_port = 80
vhost_https_port = 443
subdomain_host = frp.tanglx.cn
dashboard_addr = 0.0.0.0
dashboard_port = 11000
dashboard_user = root
dashboard_pwd = 1234@abcd
token = 770fed4ca2aabd20ae9a5dd774711de2
max_pool_count = 100
log_file = ./frps.log
log_level = info
log_max_days = 3

3.启动frps#

./frps -c ./frps.ini

四、开机自启#

a. 如Linux服务端上没有安装 systemd，可以使用 yum 或 apt 等命令安装 systemd

# yum
yum install systemd
# apt
apt install systemd

b. 使用文本编辑器，如 vim 创建并编辑 frps.service 文件。

$ vim /etc/systemd/system/frps.service

写入内容

[Unit]
# 服务名称，可自定义
Description = frp server
After = network.target syslog.target
Wants = network.target

[Service]
Type = simple
# 启动frps的命令，需修改为您的frps的安装路径
ExecStart = /path/to/frps -c /path/to/frps.ini

[Install]
WantedBy = multi-user.target

c. 使用 systemd 命令，管理 frps

 # 启动frp
systemctl start frps
# 停止frp
systemctl stop frps
# 重启frp
systemctl restart frps
# 查看frp状态
systemctl status frps
#配置开机自启
systemctl enable frps
